# Proyecto Base

Este es un proyecto base listo para personalizar.

## Estructura
- `src/`: Código fuente principal
- `tests/`: Pruebas
- `docs/`: Documentación

## ¿Cómo empezar?
Agrega tu código en la carpeta `src` y tus pruebas en `tests`.
